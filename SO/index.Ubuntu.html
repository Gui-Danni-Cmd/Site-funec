O nascimento do Ubuntu


Em 2004, Mark Shuttleworth, CEO da Canonical, uniu um time de desenvolvedores do Debian na intenção de criar uma distribuição Linux de fácil utilização para usuários comuns, eliminando diversas dificuldades que existiam na distro popular.

Desta forma, em outubro de 2004, nasceu a primeira versão do Ubuntu, de codinome Warty Warthog.
A primeira versão possuía a numeração 4.10, já que a organização das versões é baseada em mês e ano, e não em numerações seguidas.

Já o codinome, é sempre baseado em um adjetivo seguido do nome de um animal, geralmente africano, ambos iniciados com a mesma letra e obedecendo à ordem do alfabeto.

Para o que serve o Ubuntu?

Basicamente, o Ubuntu é um sistema operacional. Assim como o próprio sistema, os programas neles inseridos são livres, isto é, não há nenhum software instalado que precise ser pago ou freemium.

Os programas já vêm normalmente instalados no Ubuntu, ao contrário do Windows que só vem com os programas básicos instalados. Dessa maneira, você pode utilizar o sistema assim que o abre sem se preocupar com instalações, a não ser que queira instalar algo específico.

Muitas pessoas acham difícil utilizar alguma distribuição Linux, mas o Ubuntu é particularmente bem simples. Além disso, caso você queira conhecer a aplicação, é possível testá-la sem propriamente instalar. Legal, né?


Ubuntu e o compromisso com o open source


Um diferencial que o Ubuntu possuía nesta época era a Ubuntu Foundation, que contava com um capital pessoal investido pelo próprio Mark Shuttleworth.

Com isso, era possível enviar CDs de instalação do Ubuntu de forma gratuita para todos os usuários.

Além de facilitar o acesso à distribuição, isso ajudou a criar uma comunidade unida e engajada, algo importantíssimo para o sucesso do sistema.

O Ubuntu também conta com uma política de colaboratividade chamada “Manifesto Ubuntu“.

Nele, estão alguns valores e compromissos, como:

  - O sistema será sempre gratuito para utilizar, sem versões pagas escondidas;
  - Sempre será fácil de utilizar;
  - Sempre possuirá tecnologias assistidas e traduções para um fácil acesso
  independente do idioma do utilizador.

Na época de seu lançamento, o Ubuntu era chamado de “Debian marrom”, em um tom de crítica, por ainda estar muito próximo ao Debian, seu sistema de origem.

Apesar da primeira versão ainda ser relativamente simples, ela já contava com um instalador muito mais amigável para o usuário comum, dando opções inclusive na escolha de drivers de vídeo durante a instalação.

Esse sistema vinha com o já conhecido GNOME, mas que na época possuía um visual completamente diferente dos dias de hoje.

Ele também contava com o navegador Firefox, resultado de uma parceria que perdura até os dias de hoje.


Requisitos de sistema do Ubuntu Linux


Para que você possa instalar e usar o Ubuntu 20.04 LTS ou Ubuntu 20.10 de forma adequada é preciso que a máquina tenha os seguintes requisitos mínimos:

  - Processador dual core de 2 GHz ou superior
  - 2 GB de memória RAM
  - 25 GB de espaço livre no disco rígido
  - Uma unidade de DVD ou uma porta USB que o Ubuntu 20.04 LTS possa ser instalado
   Acesso à Internet é importante
